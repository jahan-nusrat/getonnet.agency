## <h1>Technology Used</h1>
- JavaScript
- React js
- bootstrap
- Material-UI
- HTML5
- SASS

## Code Setup
- `npm install` or `yarn install`
- `yarn start`
## Code Deployment
- Netlify
## To install Dependencies
1. `yarn add react-router-dom`
2. `yarn add react-icons`
3. `yarn add sass`
4. `yarn add styled-components`
5. `yarn add @material-ui/core`
